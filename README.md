# Time Series Data Cleaning with Master Data

## Project Structure

- **code/**: Contains all code for experiments.
  - **Experiment.java**: Entry point to run the experiments.
  - **Algorithm/**: Implementation of the core methods.
    - **MasterRepair.java**: Contains the `MasterSmooth` method.

- **data/**: Datasets used in experiments on data science applications. Details and download links for the datasets can be found in paper.

- **appendix.pdf**: Contains theoretical proofs, complete experimental results, and application evaluation.

---

## Getting Started

```bash
javac code/Experiment.java
```
